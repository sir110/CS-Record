## 条件转移指令

| 指令名称     | 说明         | 助记符  | 测试条件     |
| ------------ | ------------ | ------- | ------------ |
| 大于跳转     | 无符号数A>B  | JA/JNBE | CF=0 & ZF=0  |
| 大于等于跳转 | 无符号数A>=B | JAE/JNB | CF=0         |
| 小于转移     | 无符号数A<B  | JB/JNAE | CF=1         |
| 小于等于转移 | 无符号数A<=B | JBE/JNA | CF=1 OR ZF=1 |

**助记方法：**

J：跳转
C: 进位位置位
N: 否
S: 符号位置位
o: 溢出位置位
Z: 零标志位置位
E: 等于
P：奇偶位置位
A: Above
B: Below
L: Less (Little的比较级)
G: Greater (Great的比较级)
