## 行内与独行

* 行内公式：将公式插入到本行内
```
$ 公式内容 $
```
* 独行公式：将公式插入到新的一行内，并且居中
```
$$ 公式内容 $$
```
## 上标、下标和组合

* 上标符号：`^`
* 下标符号：`_`
* 组合符号：`{}`
